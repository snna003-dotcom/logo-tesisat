## 1. نظرة عامة

موقع تعريفي (Landing Page) من **صفحة واحدة** لمتجر مستلزمات المياه والغاز الطبيعي في مزيتلي/مرسين.

| الخاصية | التفاصيل |
|---|---|
| الهدف الأساسي | توجيه الزائر للتواصل عبر **واتساب** (14 رابط واتساب موزعة بالصفحة) |
| اللغات | تركي (افتراضي) + عربي، بزر تبديل فوري مع دعم RTL كامل |
| التقنية | HTML + CSS + JavaScript خام (Vanilla) — **بدون سيرفر، بدون قواعد بيانات، بدون build** |
| التصميم | ثيم "Obsidian & Flow" من Google Stitch: أسود فحمي `#121212` + ذهبي برونزي `#C5A059` |
| الخطوط | تركي: Playfair Display (عناوين) + Montserrat (نصوص) — عربي: Amiri + Tajawal |
| الاستضافة | GitHub Pages (مجاني دائم) + نسخة احتياطية على Netlify |

**لماذا موقع ثابت؟** لأنه لا يحتاج صيانة ولا اشتراكات، سريع جداً، آمن (لا يوجد سيرفر يُخترق)، ويعمل حتى لو فتحت ملف `index.html` مباشرة بدون إنترنت (عدا الخطوط والخريطة).

## 3. شرح ملف `index.html` (الهيكل والمحتوى)

هذا الملف يحتوي كل ما يظهر بالصفحة. مقسّم كالتالي من الأعلى للأسفل:

### أ) منطقة `<head>` — ما لا يراه الزائر لكن يهم جوجل والمتصفح

- **وسوم SEO**: `<title>` و `<meta description>` — ما يظهر في نتائج بحث جوجل.
- **Canonical + Open Graph**: عند مشاركة رابط الموقع بواتساب أو فيسبوك تظهر بطاقة أنيقة فيها صورة المتجر والعنوان — هذه الوسوم مسؤولة عنها.
- **schema.org (JSON-LD)**: بطاقة تعريف رسمية بلغة تفهمها جوجل: اسم المتجر، الهاتف، العنوان في مرسين، سنة التأسيس 2019 — تساعد بالظهور في خرائط ونتائج جوجل المحلية.
- **الخطوط**: تحميل خطوط Google Fonts الأربعة.
- **Tailwind CDN**: مكتبة تنسيقات جاهزة + إعداد ألوان الهوية (`night`, `gold`, `mist`).
- **ربط ملفات** `styles.css` والأيقونة.

### ب) جسم الصفحة — الأقسام بالترتيب

| # | القسم | معرّفه بالكود | محتواه |
|---|---|---|---|
| 1 | شريط التنقل | `<header id="navbar">` | الشعار + اسم "logo" + 5 روابط + زر اللغة + زر واتساب |
| 2 | الواجهة الرئيسية | `<section id="home">` | العنوان الكبير + وصف + زرّان + شريط إحصائيات (2019 / أفراد ومؤسسات / تجزئة وجملة) |
| 3 | المنتجات | `<section id="products">` | 8 بطاقات (مغاسل، كلوزيت، دش، فلاتر، خراطيم، قطع غيار، كومبي، غاز طبيعي) كل بطاقة فيها أيقونة + وصف + رابط واتساب |
| 4 | المعرض | `<section id="gallery">` | شبكة الصور — **تُبنى تلقائياً بالجافاسكربت** ولذلك تجدها فارغة بالـ HTML |
| 5 | من نحن | `<section id="about">` | صورة المتجر + نص تعريفي + 3 نقاط ثقة |
| 6 | تواصل | `<section id="contact">` | زر واتساب كبير + العنوان + ساعات العمل + خريطة جوجل + زر الاتجاهات |
| 7 | التذييل | `<footer>` | الشعار + روابط سريعة + العنوان + حقوق النشر |
| — | زر عائم | `<a id="waFloat">` | زر واتساب أخضر ثابت بزاوية الشاشة طوال التصفح |

### ج) ملاحظات ذكية في الـ HTML

- كل نص قابل للترجمة عليه وسم `data-i18n="مفتاح"` — الجافاسكربت يستبدله حسب اللغة.
- روابط التنقل عليها `data-nav-link` — تفعّل التمرير الناعم.
- بين الأقسام **فواصل موجية SVG** (`wave-divider`) تعطي انسيابية بصرية.
- في النهاية يُحمَّل GSAP (مكتبة الأنيميشن) ثم `main.js`.

## 4. شرح ملف `css/styles.css` (التصميم)

منظم في كتل مرقمة، أهمها:

### أ) المتغيرات (أول الملف — أهم جزء للتعديل!)

```css
:root {
  --obsidian-900: #121212;   /* لون الخلفية الأساسي */
  --gold: #C5A059;           /* اللون الذهبي المميز */
  --mist: #EDEAE3;           /* لون النصوص الفاتح */
  ...
}
```
غيّر أي لون **من هنا فقط** وسيتغير بالموقع كله.

### ب) الخطوط حسب اللغة

```css
html[lang="tr"] { --font-heading: 'Playfair Display'... }
html[lang="ar"] { --font-heading: 'Amiri'... }
```
عند تبديل اللغة تتبدل الخطوط تلقائياً لأنها مربوطة بوسم اللغة.

### ج) مكونات الواجهة

- `.glass-card` — البطاقات الداكنة الشفافة بحواف ذهبية رفيعة.
- `.btn-wa` — الزر الذهبي الأساسي (مع لمعة تعبره عند مرور الماوس).
- `.btn-chrome` — الزر الثانوي المحدد بإطار ذهبي.
- `.nav-glass` — شريط التنقل الزجاجي (يصغر تلقائياً عند النزول عبر كلاس `.shrink`).
- `.product-card` — بطاقات المنتجات مع تأثير الرفع والتوهج عند المرور.
- `.gallery-item` — صور المعرض مع التكبير والطبقة الذهبية عند المرور.
- `.lightbox` — نافذة عرض الصورة المكبرة.
- `.wa-float` — الزر العائم مع نبضة خضراء متكررة.
- `.ph` — الصورة البديلة الأنيقة إن فُقد أي ملف صورة.

### د) لمسات مهمة

- **RTL**: قواعد `[dir="rtl"]` تعكس الاتجاهات والأسهم تلقائياً للعربية.
- **الأداء**: كل الحركات تستخدم `transform/opacity` فقط (لا تجهد المعالج) وبدون blur ثقيل.
- **`prefers-reduced-motion`**: من فعّل "تقليل الحركة" بجهازه تتعطل كل الأنيميشن احتراماً له.
- **أهداف اللمس**: كل الأزرار والروابط ≥40px للاستخدام المريح بالموبايل (معيار WCAG).

## 5. شرح ملف `js/main.js` (المنطق)

مقسّم إلى 11 كتلة مرقمة بتعليقات واضحة:

### 1) قاموس الترجمة `I18N`
أكبر جزء بالملف: كائن فيه لغتان `tr` و `ar`، كل نص له مفتاح:
```js
'hero.cta2': 'Ürünleri gör'      // تركي
'hero.cta2': 'اكتشف منتجاتنا'    // عربي
```
**لتعديل أي نص بالموقع: عدّله هنا في اللغتين.**

### 2) دالة `applyLang()`
عند الضغط على زر TR|عربي:
- تغيّر `lang` و `dir` على وسم `<html>` (فينقلب الاتجاه RTL والخطوط)
- تمر على كل عناصر `data-i18n` وتستبدل نصوصها
- تحفظ الاختيار في `localStorage` فيتذكر المتصفح لغة الزائر بزيارته القادمة

### 3) أولوية الشعار الحقيقي
يجرّب تحميل `assets/logo.png` — إن وجده يستبدل النسخة المرسومة `logo.svg` تلقائياً. (لهذا يمكنك لاحقاً وضع صورة الشعار الحقيقية بهذا الاسم وستظهر فوراً).

### 4) الصور البديلة (Placeholder)
أي صورة يفشل تحميلها تُستبدل بمربع أنيق داكن بأيقونة — **الموقع لا يظهر فيه أبداً رمز الصورة المكسورة**.

### 5) خلفية الواجهة
يتأكد أن `hero.jpg` موجودة ثم يضعها خلفية، وإلا يبقى التدرج الداكن.

### 6) المعرض المرن
يجرّب تحميل `gallery-1.jpg` حتى `gallery-12.jpg`، ويعرض **ما وجده فقط** — أضف صورة برقم جديد وستظهر وحدها دون تعديل أي كود.

### 7) الـ Lightbox
عند الضغط على صورة: تفتح مكبرة بخلفية سوداء، مع أسهم تنقل، إغلاق بـ Esc أو بالضغط خارجها، وأسهم الكيبورد تعمل.

### 8) قائمة الموبايل + التمرير الناعم
زر الهامبرغر يفتح/يغلق القائمة، وروابط الأقسام تمرر بنعومة مع تعويض ارتفاع الشريط الثابت.

### 9) تصغير شريط التنقل
عند النزول أكثر من 60px يضاف كلاس `.shrink` (مقيّد بـ requestAnimationFrame للأداء).

### 10) الأنيميشن (GSAP + ScrollTrigger)
- عناصر الواجهة الرئيسية تظهر تباعاً فور التحميل.
- بقية العناصر تظهر عند وصولها بالتمرير حسب اتجاهها: `data-reveal` (من الأسفل) / `"left"` / `"right"` / `"scale"` — والاتجاهات تنعكس تلقائياً بالعربية.
- المجموعات (بطاقات المنتجات) تظهر متتابعة (stagger).
- **Parallax**: خلفية الواجهة تتحرك أبطأ من التمرير، وصورة "من نحن" تطفو عكس الاتجاه.
- **العداد**: رقم 2019 يعد تصاعدياً عند ظهوره.

### 11) سنة الحقوق
تُحدَّث تلقائياً كل سنة من تاريخ الجهاز.

## 6. شرح الواجهة (ما يراه الزائر)

1. **أول ما تفتح الصفحة**: شريط أسود زجاجي فيه الشعار الدائري واسم "logo" بذهبي، وخلفية صورة المتجر الحقيقية مع تعتيم متدرج يذوب للأسود.
2. **العنوان الرئيسي** بخط serif فاخر كبير، وكلمة "güvenilir/الموثوق" بتدرج ذهبي لامع، ثم **زرّان**: ذهبي (واتساب) ومحدد (اكتشف منتجاتنا).
3. **شريط الإحصائيات**: بطاقة داكنة فيها: تأسّس **2019** (يعد تصاعدياً) — **أفراد ومؤسسات** — **تجزئة وجملة**.
4. **مع النزول** تنساب الأقسام بفواصل موجية، وكل عنصر يظهر بحركة أنيقة.
5. **بطاقات المنتجات الثمانية**: أيقونات خطية ذهبية، وعند مرور الماوس ترتفع البطاقة وتتوهج وتعبرها لمعة.
6. **المعرض**: 9 صور حقيقية من المتجر بتخطيط متراص، عند المرور تتكبر الصورة بطبقة ذهبية وأيقونة تكبير، وبالضغط تفتح مكبرة.
7. **من نحن**: صورة المتجر بإطار ذهبي مزاح + 3 نقاط ثقة (منتجات أصلية / أسعار منافسة / أفراد ومؤسسات).
8. **تواصل**: زر واتساب أخضر كبير (أخضر عمداً — لون واتساب المعروف يرفع نسبة الضغط) + العنوان الكامل + ساعات العمل + خريطة جوجل تفاعلية + زر "الاتجاهات".
9. **طوال التصفح**: زر واتساب أخضر عائم ينبض بزاوية الشاشة.
10. **زر TR | عربي**: يقلب الموقع كاملاً بلحظة — الاتجاه، الخطوط، وكل النصوص. اسم "logo" وسلوغان العلامة يبقيان كما هما بكل اللغات (هوية تجارية).
